# Simplify

<p align="center">
  <img src="./.github/simplify-full.svg" width="480" alt="Simplify Logo" />
</p>

A typed UI design language and multi‑package design system platform for modern JavaScript and TypeScript applications.

Simplify provides a unified foundation for building responsive, deterministic, framework‑agnostic user interfaces — powered by a core compiler, design tokens, themes, UI libraries, and tooling.

## Packages

### Core
- [@simplify/engine](./packages/simplify-engine/README.md) — Typed UI design language and compiler

### UI Libraries
- @simplify/ui-react *(coming soon)* — React component library powered by the engine

### Design Language
- @simplify/tokens *(coming soon)* — Design tokens for color, spacing, typography, motion, surfaces  
- @simplify/themes *(coming soon)* — Theme system built on deterministic token resolution  
- [@simplify/brand](./packages/simplify-brand/README.md) — Official brand assets

### Tools
- @simplify/studio *(coming soon)* — Visual design tool for tokens, themes, and components  
- @simplify/playground *(coming soon)* — Live code sandbox for sheets, components, and atomic output  
- @simplify/cli *(coming soon)* — Command‑line tooling for analysis, generation, and validation
