# @simplify/brand

The official brand asset package for the Simplify Design Engine.  
This module provides the core logo files used across documentation, marketing, UI, and internal tools.

---

## Contents

This package includes two primary brand assets:

- **simplify-full.svg** — Full lockup (symbol + wordmark)  
- **simplify-icon.svg** — Icon-only version (the 8-node constellation)

Both assets are optimized SVGs suitable for use in web, app, and documentation environments.

---

## Installation

This package is part of the Simplify monorepo and is consumed internally.

```ts
import { SimplifyLogo, SimplifyIcon } from '@simplify/brand'
